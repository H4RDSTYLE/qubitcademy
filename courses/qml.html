Quantum Machine Learning

La fusión de inteligencia artificial y computación cuántica. Aprende circuitos cuánticos variacionales como modelos de ML, kernel methods cuánticos y cómo entrenar redes neuronales cuánticas con PennyLane y Qiskit.

Lo que aprenderás

  • Fundamentos de ML clásico relevantes para QML
  • Circuitos cuánticos variacionales (VQC) como modelos
  • Encoding de datos clásicos en qubits (amplitude, angle)
  • Quantum kernel methods y support vector machines cuánticos
  • Redes neuronales cuánticas (QNN) con PennyLane
  • Parameter-shift rule: gradientes en hardware cuántico
  • Quantum transfer learning
  • Clasificación cuántica de imágenes (MNIST cuántico)
  • Generative models cuánticos (QGAN, QVAE)
  • Quantum Natural Language Processing (QNLP)
  • Límites teóricos del QML: cuándo hay ventaja cuántica
  • Despliegue en hardware real con mitigación de errores

Requisitos

  • QAOA y VQE para Optimización (o conocimiento equivalente de VQC)
  • Machine learning clásico: regresión, clasificación, redes neuronales básicas
  • Python avanzado: PyTorch o TensorFlow básico, NumPy, Matplotlib
